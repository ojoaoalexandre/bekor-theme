## Installation

### Via VS Code Marketplace

1. Open VS Code or Cursor
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Bekor"
4. Click Install

### Manual Installation

1. Download the latest `.vsix` file from the [Releases](https://github.com/ojoaoalexandre/bekor-theme/releases) page
2. Open VS Code/Cursor
3. Go to Extensions view (Ctrl+Shift+X / Cmd+Shift+X)
4. Click the "..." menu and select "Install from VSIX..."
5. Select the downloaded `.vsix` file

## Activation

1. Press `Ctrl+K Ctrl+T` (or `Cmd+K Cmd+T` on Mac)
2. Select "Bekor" from the theme list

Or:
1. Go to File > Preferences > Theme > Color Theme
2. Select "Bekor"

## Color Palette

Bekor uses a warm, earthy color scheme:

- **Background**: Deep brown (#1A1612)
- **Foreground**: Warm beige (#E8DCC8)
- **Accent**: Golden yellow (#D4AF37)
- **Syntax**: Carefully chosen colors for different code elements

## Language Support

Bekor provides syntax highlighting for all languages supported by VS Code/Cursor, including:
- JavaScript/TypeScript
- Python
- Java
- C/C++
- HTML/CSS
- JSON
- Markdown
- And many more!
